# vue-cli

## 技术栈
- vue2 + vuex + vue-router + webpack + ES6 + axios + stylus

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
